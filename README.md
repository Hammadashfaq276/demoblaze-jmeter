This project demonstrates performance testing of the DemoBlaze e-commerce web application using Load, Stress, Spike, and Endurance Testing techniques.

The purpose of this project is to evaluate application behavior under different load conditions, ensuring reliability, scalability, and stability.

This project is created for portfolio, learning, and SDET practice purposes.
